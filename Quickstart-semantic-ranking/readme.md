# Quickstart: Semantic ranking in Azure AI Search using REST APIs

![Flask sample MIT license badge](https://img.shields.io/badge/license-MIT-green.svg)

This sample demonstrates how to set up semantic ranking. You add a semantic configuration to a search index, and then you add semantic parameters to a query.

## What's in this sample

| File | Description |
|------|-------------|
| `semantic-index-update.rest` | REST file containing requests to add a semantic configuration to an existing index |
| `semantic-query.rest` | REST file containing semantic ranking queries with captions and answers |

## Documentation

This sample accompanies [Quickstart: Semantic ranking using REST](https://learn.microsoft.com/azure/search/search-get-started-semantic?pivots=rest). Follow the documentation for prerequisites, setup instructions, and detailed explanations.
