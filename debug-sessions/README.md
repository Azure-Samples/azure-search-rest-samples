# Fix skillset issues using Debug Sessions in Azure AI Search

![Flask sample MIT license badge](https://img.shields.io/badge/license-MIT-green.svg)

This sample creates a "buggy" enrichment pipeline that you can fix using the Debug Sessions visual editor in the Azure portal.

## What's in this sample

| File | Description |
|------|-------------|
| `debug-sessions.rest` | REST file containing requests to create a data source, indexer, skillset, and index with intentional issues for debugging |
