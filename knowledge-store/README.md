# Create a knowledge store in Azure AI Search using REST APIs

![Flask sample MIT license badge](https://img.shields.io/badge/license-MIT-green.svg)

This sample creates a knowledge store with AI-generated content from image and text analyses, stored durably in Azure Storage for consumption by other processes or apps.

## What's in this sample

| File | Description |
|------|-------------|
| `knowledge-store.rest` | REST file containing requests to create a data source, indexer, skillset with knowledge store projections, and index |

## Documentation

This sample accompanies [Create a knowledge store using REST](https://learn.microsoft.com/azure/search/knowledge-store-create-rest). Follow the documentation for prerequisites, setup instructions, and detailed explanations.
